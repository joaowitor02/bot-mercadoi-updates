# Guia de Instalação — Plugin Bot Mercadoi API

**Tempo estimado:** 5 minutos  
**Nível:** Nenhum conhecimento técnico necessário

---

## O que é este plugin?

Este plugin cria uma conexão segura entre o bot e o seu site WordPress.  
Com ele instalado, o bot publica os imóveis **diretamente** no seu site — sem precisar abrir o navegador manualmente.

---

## Passo 1 — Receber o arquivo

Você receberá um arquivo chamado **`bot-mercadoi-api.zip`**.  
Guarde-o em um lugar fácil de encontrar (ex: Área de Trabalho).

---

## Passo 2 — Instalar o plugin

1. Acesse o painel do seu WordPress:  
   `https://seusite.com.br/wp-admin`

2. No menu lateral, clique em **Plugins → Adicionar novo**

3. Clique no botão **Enviar plugin** (no topo da página)

   ![Botão Enviar plugin](https://i.imgur.com/placeholder.png)

4. Clique em **Escolher arquivo**, selecione o `bot-mercadoi-api.zip` e clique em **Instalar agora**

5. Após a instalação, clique em **Ativar plugin**

---

## Passo 3 — Gerar a chave de API

1. No menu lateral do WordPress, clique em **Ferramentas → Bot Mercadoi API**

2. Clique no botão **Gerar chave de API**

3. Uma sequência de letras e números vai aparecer, por exemplo:  
   `a3f7c2e1b9d4...`

4. **Copie essa chave** e me envie — ela é necessária para o bot se conectar ao seu site

> ⚠️ **Atenção:** Não compartilhe essa chave com ninguém além do bot. Ela dá acesso de publicação ao seu site.

---

## Passo 4 — Me enviar as informações

Após gerar a chave, me mande:

- ✅ A **chave de API** gerada no passo anterior
- ✅ A **URL do seu site** (ex: `https://mercadoi.com.br`)

Só isso! Com essas duas informações faço a configuração no bot e já podemos testar.

---

## Perguntas frequentes

**O plugin muda alguma coisa no visual do site?**  
Não. Ele funciona nos bastidores e não interfere em nada que os visitantes vejam.

**E se eu precisar revogar o acesso do bot?**  
Basta ir em Ferramentas → Bot Mercadoi API e clicar em **Remover chave**. O bot para de funcionar imediatamente.

**Posso desinstalar o plugin depois?**  
Sim. Os imóveis já publicados ficam no site normalmente. O plugin só é necessário para novas publicações.
